<h1 align="left">
    HeLIX
</h1>

HeLIX is a ui library for Unity UI Toolkit that provides a collection of customizable and reusable
UI components, styles, and utilities to streamline the development of user interfaces in Unity applications.

## Installation

You can install HELIX Core via Unity Package Manager by adding the following Git URL:
`https://github.com/helightdev/HELIX.git?path=src/HELIX/Assets/Plugins/HELIX`

### Widgets
`https://github.com/helightdev/HELIX.git?path=src/HELIX/Assets/Plugins/HELIX.Widgets`
